# لعبة توقعات كأس العالم 2026

برنامج ويب جاهز للنشر على GitHub Pages.

## التشغيل محلياً
افتح ملف `index.html` في المتصفح.

## النشر على GitHub Pages
1. أنشئ مستودعاً جديداً في GitHub.
2. ارفع جميع الملفات الموجودة في هذا المجلد إلى جذر المستودع.
3. افتح:
   `Settings → Pages`
4. اختر:
   `Deploy from a branch`
5. اختر الفرع:
   `main`
6. اختر المجلد:
   `/root`
7. اضغط `Save`.

بعد النشر سيظهر رابط الموقع في صفحة GitHub Pages.

## الملفات المهمة
- `index.html`: البرنامج الرئيسي.
- `wc2026_logo.png`: الشعار.
- `404.html`: يعيد فتح البرنامج إذا تم الوصول إلى رابط غير مباشر.
- `.nojekyll`: يمنع GitHub Pages من معالجة الملفات بواسطة Jekyll.
- `README.md`: تعليمات النشر.

## الحفظ
البرنامج يحفظ البيانات داخل متصفح المستخدم باستخدام Local Storage.
استخدم زر النسخة الاحتياطية داخل البرنامج لحفظ ملف JSON واستعادته لاحقاً.


## تحديث جدول الترتيب النهائي
تمت إضافة:
- عدد المباريات التي توقعها اللاعب بشكل صحيح.
- عدد توقعات الكروت الصحيحة لكل لاعب.

توقع الكروت الصحيح يُحسب لكل فريق على حدة إذا كان عدد الأصفر والأحمر مطابقاً للنتيجة الحقيقية.


## إصلاح إحصائيات جدول الترتيب
تم فصل الإحصائيات إلى:
- مباريات توقعها صح: توقع الفائز/التعادل في المجموعات أو الفائز النهائي في الإقصائيات.
- نتائج دقيقة صحيحة: توقع نتيجة 90 دقيقة بالضبط.
- توقعات كروت صحيحة: تُحسب لكل فريق على حدة عند مطابقة الأصفر والأحمر.


تحديث: تم تحويل إدخال توقعات المباريات إلى اختيار مباراة ثم إدخال توقعات جميع المشتركين في صفحة واحدة.


تحديث الواجهة: كل مباراة تعرض الفريق الأول في جهة والفريق الثاني في الجهة المقابلة، وتحت اسم كل فريق حقول الأهداف والكروت الصفراء والحمراء.
